qwerty drum machine

 

github    .zip


‘qwerty drum machine’ is a simple drum machine program written in C++ with openFrameworks and the Maximilian add-on. There are two drum effects possible – one for a kick drum type sound and on for a snare. There is also some radial sound visualisation so that one can see visually what is happening to the soundwaves as they play. The drum machine works on eight beats to the bar, and individual hits can be triggered by using the computer keyboard, with the ‘qwerty..’ row for the snare and the ‘asdf..’ for the kick. The kick is created with an enveloped sine wave, with additional modulation possible via FM synthesis methods. There are controls for the frequency of the carrier wave, the frequency of the modulating wave, the modulation amount as well as ADSR and volume controls. The snare is created simply with a filtered and enveloped noise wave, using of Maxim's built in envelope and filter functions. There are controls for filter cut-off, ADSR and volume. There is also a global tempo control, which is kept in time by a phasor oscillator and some counting variables.
