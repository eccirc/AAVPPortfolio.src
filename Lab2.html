‘noise ripple’ is an image processing program written in C++ with openFrameworks. At it’s core is a lateral stretching algorithm, then the ripple effect is added using OF’s built in perlin noise function. There are controls for lateral offset and the RGB offset, the scale of the noise wave (essentially how deep the peaks and troughs), the noise multiplier (how tight the ripple appears) and for adding a second dimension of noise to the vertical axis.
